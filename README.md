# 💰 Money Tracker

A simple personal finance tracker built with HTML, CSS, and JavaScript. It helps users record daily income and expenses, organize transactions, and keep track of their financial activities in one place.

## 🌐 Live Demo

https://najmulsprofile.github.io/Money-Tracker-App/

## 📱 Download APK

Download the latest Android APK from the **Releases** section:
https://github.com/najmulsprofile/Money-Tracker-App/releases/tag/v1.0.0

## ✨ Features

* Track income and expenses
* View your current balance
* Manage transaction history
* Edit and delete transactions
* Clean and responsive interface
* Local storage (no account required)
